3. Mobile-First Approach

Tailwind uses mobile-first responsive design:

class="text-center md:text-left lg:text-right"

This text alignment changes:

Mobile: Center | Tablet: Left | Desktop: Right

How it works: 

text-center     → All screens (mobile first)
md:text-left    → Medium screens and up (≥768px)
lg:text-right   → Large screens and up (≥1024px)
